Бот будет добавлять, показывать и отмечать задачи как выполненные. 

Пример: 

-/start -Добро пожаловать! Я могу организовать ваши задачи. 
Команды:
 /add-добавить задачу 
/tasks-показать список задач 
/done-выполнено 
/donedTasks-список выполненных задач
/delete-удалить 
/help-помощь

 Полить цветы

 Формулировка третьей задачи 

Предоставление пользователю возможности выгрузить из тг бота(экспортировать) и импортировать список своих задач (текущих и выполненных) через JSON файл. 
Задачи при импорте должны дополнить уже имеющиеся списки в боте
При совпадении задач из списка и из файла, бот убирает повторы и оставляет только одну запись.


В список всех команд добавляются  /export и /import
/export-предоставить список задач пользователя в файле
/import-загрузить список задач из файла

Содержимое файла: сначала текущие задачи, далее выполненные

Пример:
Предположим, что у нас есть списки команд:
Вот список ваших задач:
1.Полить цветы
2.Накормить кошку
Вот список выполненных задач:
1.Помыть пол

Пользователь: -/export “tasks_list”
Бот: -Ваш список задач в виде документа (отправка “tasks_list.json”)

Пример:
Например загружаем тот же файл «tasks_list.json”
Пользователь: /import
Бот: -Отправьте JSON файл со списком задач
Пользователь отправляет файл и его задачи из current_tasks добавляются в список текущих задач, а completed_tasks в список выполненных.
При совпадении задач из списка и из файла, бот убирает повторы и оставляет только одну запись.
Бот: -Задачи успешно добавлены, можете проверить списки с помощью команд /tasks и /dTask
При ошибке: -Что-то пошло не так, отправьте файл еще раз или проверьте его содержимое.


Пример содержимого файла:
*******************tasks_list.json****************************
{
	“current_tasks”: [
		“Полить цветы”,
		“Накормить кошку”
	],
	“completed_tasks”: [
		“Помыть пол”
	]
}
********************************************************************
current_tasks - список текущих задач пользователя
completed_tasks - список выполненных задач пользователя





-/delete Накормить кота
-Задача “Накормить кота” удалена из задач
