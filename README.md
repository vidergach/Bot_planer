Формулировка второй задачи 

Ежедневник с задачами 
Бот будет добавлять, показывать и отмечать задачи как выполненные. 

Пример: 

-/start -Добро пожаловать! Я могу организовать ваши задачи. 
Команды:
 /add-добавить задачу 
/tasks-показать список задач 
/done-выполнено 
/dTask-список выполненных задач
/delete-удалить 
/help-помощь

 -/add Полить цветы 
-Задача добавлена 

-/add Накормить кота
-Задача добавлена

-/tasks
-Вот список задач:
 Полить цветы
Накормить кота 

-/done Полить цветы
-Задача “Полить цветы” отмечена выполненной!

-/dTask
-Вот список выполненных задач:
 Полить цветы

-/delete Накормить кота
-Задача “Накормить кота” удалена из задач
